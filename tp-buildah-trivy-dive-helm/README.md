# TP — Buildah, Trivy, Dive & Helm/Kubernetes — MIAGE Bank

## Rendu

Le rendu se fait **exclusivement par Pull Request GitHub** ciblant la branche `main`.

### Comment soumettre votre travail

1. Forkez ce dépôt ou créez une branche depuis `main`.
2. Dans ce dossier (`tp-buildah-trivy-dive-helm/`), créez un sous-dossier au nom de votre groupe :
   ```
   tp-buildah-trivy-dive-helm/
   └── prenom1-nom1_prenom2-nom2/
       ├── README.md   ← doit mentionner vos noms en première ligne
       └── ...         ← vos livrables (Partie A et Partie B)
   ```
3. Ouvrez une Pull Request vers la branche `main` de ce dépôt.

### Contraintes

- Le `README.md` à la racine de votre dossier doit obligatoirement mentionner votre **nom et prénom** en première ligne.
- Les groupes sont limités à **2 personnes**.
- Date limite de rendu : **05 juin**. Toute PR ouverte après cette date ne sera pas prise en compte.
